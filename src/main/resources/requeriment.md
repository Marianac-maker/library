## Requisitos 

Quiero crear un API rest en Java superior a 23 utilizando el framwork spring bot y lambok , cuyo objetivo sea gestionar un sistema integral de reservas y servicios para un parque ecológico . El sistema debe contemplar la ADMI de visitantes, reservas, actividades , guías , pagos y servicios adicionales , garantizando la persistencia de los datos en un archivo separado por comas (CSV). Quiero manejar nombramientos en inglés , conservando principios SÓLID, KISS y DRY, persistiendo la información en archivos separados por comas. . Dentro de la base modelo pon las clases Persona,
Usuario ,Autor ,Libro,Categoria,Editorial,Prestamo,Devolucion,Reseña,RegistroMulta.
